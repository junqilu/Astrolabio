# Astrolabio
Named after the astrolabe, the instrument once used to navigate uncharted territories, Astrolabio is a bioinformatics project that maps astrocyte transcriptional states across health and disease. By integrating multiple RNA-seq datasets from diverse biological contexts, Astrolabio serves as a navigational tool for identifying genes and pathways of interest, helping researchers chart promising directions for future astrocyte research.
